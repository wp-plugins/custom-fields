== Description ==

This plugin add custom fields for some things on WordPress, blog, term taxonomy and custom object types

== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. Activate the plugin within you WordPress Administration Backend
3. Go to Settings > Custom Fields and follow the steps on the [Custom Fields](http://redmine.beapi.fr/projects/show/custom-fields) page.
